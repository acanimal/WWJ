Description:
------------
This is only an exercice trying to create better annotations using JPanel. This way we have all the 
power of Swing to create our annotations.

There are to requirements:
- Use at least JDK6u12 (currently at early access).
- The container on which WWJ Canvas resides must use a null layout. See the example.


Changes from original WWJ 0.5.0:
--------------------------------
No change on the original source code. All you need is to copy the files to the specified packages.
